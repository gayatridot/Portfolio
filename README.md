# Portfolio2

This is a personal portfolio website project. It showcases education, experience, skills, certificates, and projects, and includes a custom CSS stylesheet and images.

## Project Structure

- `index.html` — Main landing page
- `Edu.html` — Education details
- `Experience.html` — Work experience
- `skill.html` — Skills overview
- `certificates.html` — Certificates and achievements
- `project.html` — Project portfolio
- `portfolio.css` — Stylesheet for the website
- `images/` — Image assets (e.g., Adobe cloud service.jfif)

## How to Use

1. Open `index.html` in your browser to view the main page.
2. Navigate to other sections using the provided links.
3. Customize the HTML and CSS files to update your personal information, style, or add new sections.

## Customization
- Replace images in the `images/` folder with your own.
- Edit the HTML files to update your content.
- Modify `portfolio.css` to change the look and feel.
